Web-based event mappping

The Wedding Expo used to map their events on paper. The event mapping tool integrates all exhibitor application data, allowing for up-to-date accuracy.

It's a customized Leaflet.js map built on a Rails app, with a friendly mobile read-only version for on-the-ground use. There's also a pdf export tool built with Puppeteer.
